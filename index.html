BSM

Bundled Software Manager

Get Started

Manage Packages

BSM was designed to be a package manager for projects with amounts of packages. The initial purpose for BSM is to simplify the management of High Energy Physics software.

Build Application

Build your own application with BSM framework. BSM could help you with most common configurations and operations, also provide the ability to customize the application.

Extendible with Python

Extend the functionality of your application with Python. The behavior could be fine controlled with the Python handler.
